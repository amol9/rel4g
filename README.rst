=====
rel4g
=====


Command line interface to Reliance 4G Wi-Pod web interface. Also, a bandwidth usage monitoring service.


Supported platforms
===================

* Python 2.7 / 3.x
* Linux


Features
========



Installation
============

Install via pip::
    
    pip install rel4g
